## 摘要
超维计算（HDC）是一种有前景的能效边缘机器学习方法，其中低延迟、低功耗和紧凑的内存预算至关重要。然而，传统的HDC依赖于符号绑定和伪随机高维向量，这需要较大的维度和启发式更新以达到有竞争力的准确率，从而限制了在边缘硬件上的部署。我们提出了XL-HD，一种确定性的、基于投影的、完全可学习的HDC框架，专为边缘计算系统中的内存加速而设计。该方法使用固定的Sobol序列投影二进制输入，将学习扩展到传统HDC之外。在训练过程中，类原型在实值空间中进行优化，随后进行二值化，从而实现完全二值化的点积推理流水线，非常适合用于如ReRAM交叉阵列等IMC硬件。XL-HD在MNIST、UCIHAR和ISOLET上取得了有竞争力的准确率，同时保持了紧凑的基于IMC的推理引擎，面积为0.395 mm²，单周期推理仅需0.40 μJ。

## Abstract
Hyperdimensional computing (HDC) is a promising approach for energy-efficient edge machine learning (ML), where low latency, low power, and tight memory budgets are essential. However, traditional HDC relies on symbolic binding and pseudo-random high-dimensional vectors, which require large dimensionality and heuristic updates to reach competitive accuracy, limiting deployment on edge hardware. We introduce XL-HD, a deterministic, projection-based, fully learnable HDC framework tailored for in-memory acceleration within edge computing systems. The method uses a fixed Sobol sequence to project binary inputs, extending learning beyond conventional HDC. During training, class prototypes are optimized in real-valued space and later binarized, enabling an entirely binary dot-product inference pipeline ideal for IMC hardware such as ReRAM crossbars. XL-HD achieves competitive accuracy on MNIST, UCIHAR, and ISOLET while maintaining a compact IMC-based inference engine with $0.395 \ \text{mm}^2$ area and only $0.40 \ μ\text{J}$ per single-cycle inference.

## 1. 核心问题与研究动机

- **问题背景**：超维计算（HDC）是一种轻量级的边缘机器学习范式，具有低延迟、低功耗、容错等优势，适合部署在资源受限的边缘设备上。传统HDC依赖伪随机高维向量（hypervectors）和符号绑定、累积等操作，需要较高的维度（通常4K-8K）以保证分类精度，且更新过程是启发式的，缺乏可学习性。
- **核心挑战**：
  - **维度高导致存储和计算开销大**：低精度（1-bit）HDC需要大幅提升维度才能保持精度，不利于面积和能耗。
  - **伪随机向量分布不均匀**：需要更大维度来保证正交性，限制了硬件效率。
  - **现有HDC框架缺乏端到端硬件映射**：许多方法仍需要多周期、混合精度或外部计算，难以实现单周期、全二进制推理。
- **研究目标**：提出一种确定性的、完全可学习的HDC框架，结合内存计算（IMC）硬件（如ReRAM交叉阵列），实现低维度、单周期、低能耗的边缘推理。

## 2. 方法论：核心思想与技术细节

- **核心思想**：用确定性低差异序列（Sobol序列）构造投影矩阵，替换传统伪随机投影；在训练阶段将类原型学习建模为可微的余弦相似度+交叉熵损失优化（实值空间），推理时二值化原型和查询超向量，实现完全二进制点积运算，映射到IMC交叉阵列。

- **关键技术细节**：
  1. **确定性Sobol投影**：生成固定投影矩阵 \( \mathbf{P} \in \mathbb{R}^{D \times F} \)，对Sobol序列元素阈值化（>0.5→+1，否则→-1），得到双极性矩阵 \( \mathbf{P} \in \{-1, +1\}^{D \times F} \)。输入特征 \( \mathbf{x} \in \mathbb{R}^F \) 通过线性映射 \( \mathbf{h} = \mathbf{P}\mathbf{x} \) 得到实值超向量。
  2. **可学习的类原型**：初始化为同类样本编码超向量的累积（centroid），然后通过余弦相似度 + Softmax + 交叉熵损失，使用梯度下降（Adam等）在实值空间更新原型矩阵 \( \mathbf{C} \in \mathbb{R}^{K \times D} \)。训练结束后，将原型二值化（阈值0或符号函数）以匹配硬件。
  3. **推理流水线**：查询样本按同样Sobol投影 → 实值超向量。在硬件中，利用双交叉阵列（P+, P-）实现差分电流比较进行“原位二值化”（式2），得到二进制查询超向量 \( \mathbf{h}_q \in \{0,1\}^D \)。然后与存储在分类器交叉阵列中的二值化类原型进行二进制点积（式3），通过比较器树选出最大相似度类。

- **算法流程（文字说明）**：
  - **训练阶段**（算法1）：
    1. 固定Sobol投影矩阵P，并二值化。
    2. 对每个训练样本，计算编码超向量 \( h_i = P x_i \)。
    3. 按类别累积初始化原型 \( C_k \)。
    4. 归一化原型后，使用小批量梯度下降（cosine相似度+交叉熵损失）更新原型 \( C \)。
  - **推理阶段**：
    1. 对查询 \( x_q \) 计算 \( h_q = P x_q \)，原位二值化。
    2. 计算 \( z_k = \text{popcount}(h_q \odot c_k) \)（二进制点积）。
    3. 选择最大 \( z_k \) 作为预测类。
