## 摘要
神经动力系统是具有表达力的时间预测器，通过细粒度状态更新捕获连续时间动态。然而，这种顺序结构难以映射到专为密集矩阵运算优化的数字硬件上，而具有原生连续时间动态的模拟神经形态计算可以解决这种不匹配。我们介绍了FerroNDS，一种由两个模拟原语构建的神经形态系统：用于时间累积的积分器和用于频率选择性滤波的振荡器。我们将该系统映射到基于多比特铁电二极管的存内计算硬件上。一个128神经元的FerroNDS实例计算短时傅里叶变换，并预测周期、准周期和混沌信号的500毫秒时域。该系统实现了亚瓦特实时运行，每个神经元每次推理的能量为1.64 μJ（200 Hz）和0.29 μJ（10 kHz），比基于SRAM的数字系统面积减少25-40倍，每层延迟为3.18 ms（200 Hz）和63.87 μs（10 kHz）。据我们所知，这是首次将铁电二极管端到端集成到神经形态计算框架中，确立了铁电存内计算作为模拟神经动力系统的实用基底。

## Abstract
Neural dynamical systems are expressive temporal predictors that capture continuous-time dynamics through fine-grained state updates. However, this sequential structure maps poorly onto digital hardware optimized for dense matrix operations, a mismatch that analog neuromorphic computing, with its native continuous-time dynamics, can resolve. We introduce FerroNDS, a neuromorphic system built from two analog primitives: an integrator for temporal accumulation and an oscillator for frequency-selective filtering. We map this system onto compute-in-memory hardware based on multi-bit ferrodiodes. A 128-neuron instance of FerroNDS computes short-time Fourier transform and forecasts a 500-ms horizon for periodic, quasi-periodic, and chaotic signals. The system achieves sub-watt real-time operation with per-neuron per-inference energy of 1.64 $μ$J (200 Hz) and 0.29 $μ$J (10 kHz), 25-40$\times$ area reduction over SRAM-based digital systems, and per-layer latency of 3.18 ms (200 Hz) and 63.87 $μ$s (10 kHz). To our knowledge, this is the first end-to-end integration of a ferrodiode into a neuromorphic computational framework, establishing ferroelectric compute-in-memory as a practical substrate for analog neural dynamical systems.

- **研究背景**：模拟电路可直接实现相同微分方程，时间作为物理量演化而非循环索引，天然解决该不匹配。神经形态计算长期以来将微分方程映射到漏积分神经元和静态非线性函数网络上。振荡器可完成基于频率的任务（如傅里叶分析），积分器可完成时间累积任务（如从加速度确定位置和速度）。这两种操作构成 Izhikevich 神经元模型的基础，且模拟电路在低精度下更高效。此外，存内计算（CIM）可减少通信开销，实现数十至数百毫瓦功耗，远优于单板计算机（如 Jetson Nano 数瓦）。
- **整体含义**：本文首次将多比特铁电二极管（FeD）与模拟振荡器/积分器动力学端到端整合，提出 FerroNDS 系统，证明铁电存内计算可作为模拟神经动力系统的实用基底，实现实时、低功耗信号预测。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 2.1 核心思想
- 利用两个模拟原语：**积分器（Leaky Integrator, LI）** 用于时间累积，**振荡器（Bandpass Oscillator, BP / Undamped Harmonic, UH）** 用于频率选择性滤波。
- 将神经动力系统映射到基于多比特铁电二极管（FeD）的存内计算硬件上，通过设备-电路-算法协同设计实现端到端的实时预测。

### 2.2 关键技术细节

#### (a) 多比特铁电二极管（FeD）及其线性化电路
- FeD 特性：20-nm AlScN 薄膜，5-bit 操作，10 fJ/bit 开关能量，自整流 I-V 特性消除了对选择元件的需求。
- 线性化电路：5 级电路（缩放放大器、跨阻放大器、感测放大器、对数放大器、反相求和器）将 FeD 的指数 I-V 特性转换为线性输入-输出关系，使得每个 FeD 状态对应一条直线 `y = m_i * x + b_i`。
- 状态选择算法：从 32 个状态中筛选出 8 个满足四个约束（最小电流水平、指数拟合质量、A_i 系数线性间隔、G_i 系数均匀性）的状态，工作电压范围 [6.45, 7.45] V。

#### (b) 积分器与振荡器动力学
- **积分器（LI）**：基于一阶漏积分模型：`dy/dt = (x - y)/τ`。离散形式：`y_k = α*y_{k-1} + (1-α)*ϕ(x_{k-1})`，其中 `α = σ(e^{-Δt/τ})`，`ϕ` 为带权重的输入。电路实现包括充电-放电突触模块和漏积分体细胞电路。
- **振荡器（BP）**：基于谐振-发火（RF）神经元导出的带通振荡器：`dz1/dt = -2ξω z1 - ω^2 z2 + x`, `dz2/dt = z1`。离散形式通过欧拉积分。电路实现利用 LC 槽路和运放反馈，谐振频率由 L、C 决定，可实现 200 Hz。阻尼系数可调，响应呈现带通滤波特性。
- **无阻尼振荡器（UH）**：用于长时程依赖，采用隐式-显式（IMEX）离散化避免振幅误差，适用于混沌信号（如 Mackey-Glass）。

#### (c) 网络架构与算法流程
- **BP/LI 架构（前馈）**：先通过 64 个 BP 振荡器（带可学习递归连接）将输入投影到傅里叶类特征空间，再经 64 个 LI 单元进行时间池化，最后线性读出。所有参数端到端训练。
- **Reservoir BP 架构**：128 个全连接 BP 振荡器池，线性读出，所有权重（内部和读出）端到端学习。
- **UH/LI 架构**：堆叠 UH 和 LI 模块，加入门控线性单元（GLU）和 GELU 激活，用于混沌信号预测。训练参数包括投影矩阵、频率、时间常数等。
- 训练优化器：Adam，损失函数：MSE。无正则化或梯度裁剪。

## 6. 主要结论与发现
1. **首次端到端整合 FeD 到神经形态计算框架**：FerroNDS 证明了基于铁电存内计算的模拟神经动力系统在实时预测中的可行性。
2. **架构优势**：振荡器群体自然产生短时傅里叶变换（STFT）行为，将非平稳信号转化为稀疏频谱特征，利于预测；无阻尼振荡器（UH）可捕获长程依赖。
3. **低功耗高性能**：128 神经元系统在 200 Hz 下每神经元每推理能量 1.64 μJ，10 kHz 下 0.29 μJ，面积较 SRAM 缩小 25-40 倍，每层延迟 3.18 ms（200 Hz）或 63.87 μs（10 kHz）。在亚瓦特功率预算内（1 W 下支持 3000 单元），适合电池供电边缘节点。
4. **预测精度**：500-ms 时域预测 MSE 在周期信号 ≤0.046，准周期信号 ≤0.319，混沌信号 ≤0.003。显著优于 MLP 基线（MSE 可达 0.655）。
5. **多比特 vs 二进制**：多比特 FeD 电路在 3 比特以上精度时更节能（约 2× 优于二进制）。

## 7. 优点：方法与实验设计亮点
- **设备-电路-算法端到端协同设计**：从 FeD 器件特性出发，设计线性化电路、状态选择算法，再映射到离散 NDS 网络，形成完整流程，具有通用性（可适配其他多比特 eNVM）。
- **电路设计与数学建模紧密结合**：积分器和振荡器电路使用标准 SPICE 仿真，数学离散化与硬件实现一致（欧拉积分、IMEX）。
- **引入无阻尼振荡器（UH）用于长时程预测**：采用 IMEX 离散化避免了传统欧拉方法的不稳定问题，在混沌信号 Mackey-Glass 上取得极低误差。
- **系统性基准对比**：不仅对比常用 MLP，还在不同架构（前馈、储备池、UH）之间比较，涵盖周期、准周期、混沌三类信号。
- **硬件基准指标全面**：能量、面积、写入能量、延迟均对标现有商用神经形态芯片（Loihi、Xylo-A2）及传统数字系统，并以表格和图示清晰展示。
